### Шаг 2: Установка на сервер
🖥️ **Следуйте**: [INSTALLATION.md](INSTALLATION.md)
- Выбирите вашу ОС (Linux?)
- Установка в production

### Шаг 3: Безопасная конфигурация
🔒 **Конфигурируйте**:
```bash
# Используйте переменные окружения для пароля
export IMAP_PASSWORD=your_secure_password

# НЕ коммитьте config.yaml с паролем в git!
```

### Шаг 4: Запуск как сервис
🚀 **Запустите** с production сервером:
```bash
# Вместо Flask развертывания используйте gunicorn
pip install gunicorn
gunicorn -b 0.0.0.0:5555 data.app:app
```

### Шаг 5: Мониторинг и автоматизация
📊 **Настройте**:
- Ежедневное обновление через cron/systemd
